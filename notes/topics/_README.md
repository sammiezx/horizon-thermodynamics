# Topic notes

Build a file here once you have ≥ 3 per-paper notes that touch the same idea.
A topic note is **the durable form** of your understanding — it should make sense
without re-reading any of the papers.

Suggested first topic notes (delete as you create real ones):

- `hawking_derivation.md` — Hawking's original semiclassical argument, plus modern
  re-derivations (tunneling, anomaly, Euclidean).
- `page_curve.md` — Page's information argument and what islands changed.
- `greybody_factors.md` — what they are, how they distort the thermal spectrum,
  how to compute them.
- `analog_horizons.md` — what counts as a horizon in BEC/fluid/optical systems,
  what's been observed.
- `qft_curved_spacetime.md` — Bogoliubov coefficients, vacuum ambiguity,
  particle creation as basis change.
