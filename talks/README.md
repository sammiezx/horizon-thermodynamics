# Talks

One subfolder per talk. Use Beamer (LaTeX) for journal-club / group-meeting
slides — keeps math notation consistent with your papers and lets you reuse
the master `references/library.bib`.

```
talks/
├── YYYY-MM-DD-short-title/
│   ├── slides.tex
│   └── figures/
```

Naming convention: `YYYY-MM-DD-short-title` (date prefix sorts chronologically).
