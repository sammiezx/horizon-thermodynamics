# Papers

PDFs of papers you read. Drop them in the topic folder that fits best — if a paper
spans topics, pick the dominant one, and link to it from the others' notes.

## Conventions

- **Filename:** `<CiteKey>.pdf` matching the BibTeX key in
  [../references/library.bib](../references/library.bib).
  Example: `Hawking1975.pdf`, `PenningtonShenkerStanfordYang2019.pdf`.
- **One PDF per paper.** If you have v1 / v2 / published, keep only the most recent
  unless the differences matter.
- **Notes go in [../notes/per_paper/](../notes/per_paper/)**, not here. Keep this
  directory PDFs-only.

## Folders

- `hawking_radiation/` — Hawking's derivation, semiclassical gravity, mode-mixing,
  thermal spectrum, Unruh effect.
- `information_paradox/` — Page curve, AMPS firewalls, ER=EPR, islands, replica
  wormholes, complementarity.
- `numerical/` — greybody factors, mode functions, late-time spectra, numerical
  relativity treatments of evaporation.
- `analog_gravity/` — BEC, sonic, optical, fluid analog horizons; experimental papers.
- `foundations/` — GR textbooks/papers, QFT-in-curved-spacetime references, AdS/CFT
  background, holography basics.
